## 📦 Добавленные функции

### 1. CSS Анимации и Переходы

#### Кнопки (Buttons)
- **Hover-эффекты**: поднятие на 2px с увеличением тени
- **Active-состояние**: небольшое нажатие вниз
- **Плавные переходы**: cubic-bezier(0.23, 1, 0.320, 1) для естественных движений
- **Различные стили**: primary, secondary, ghost, danger, link

**Код:**
```css
.btn:hover {
  transform: translateY(-2px);
  box-shadow: 0 8px 20px rgba(22, 163, 74, 0.35);
}
```

#### Карточки (Cards)
- **Entrance animation**: fadeInUp с задержкой
- **Hover-эффект**: лифт на 4px с изменением цвета бордера
- **Shine-эффект**: градиент光 на hover
- **Каскадное появление**: каждая карточка появляется с задержкой 0.1s

**Пример анимации:**
```css
@keyframes fadeInUp {
  from { opacity: 0; transform: translateY(12px); }
  to { opacity: 1; transform: translateY(0); }
}
```

#### Навигация
- **Underline animation**: полоса под пунктом меню появляется слева направо
- **Color transition**: плавное изменение цвета при hover
- **Активный пункт**: полная полоса под текущей страницей

#### Галерея
- **Image zoom**: масштабирование изображения на 1.08x при hover
- **Glow effect**: радиальный градиент появляется при hover
- **Каскадное появление**: каждое изображение появляется с интервалом 0.1s

### 2. Интерактивная Галерея с Модалью

#### Функции:
- 🔍 **Клик на изображение**: открывает модальное окно с полноразмерным изображением
- ⬅️➡️ **Навигация**: стрелки для переключения между изображениями
- ⌨️ **Клавиатурная навигация**: 
  - Стрелки влево/вправо для переключения
  - Escape для закрытия
- 🖱️ **Клик по фону**: закрывает модальное окно
- 📊 **Счетчик**: показывает текущее изображение и общее количество
- 📝 **Подпись**: отображает названи изображения из figcaption

#### CSS Стили Модального Окна:
```css
.modal {
  animation: fadeInUp 0.4s ease;
  box-shadow: 0 20px 60px rgba(0, 0, 0, 0.3);
  border-radius: 20px;
}

.modal-overlay {
  backdrop-filter: blur(4px);
}
```

### 3. Улучшенная Форма Обратной Связи

#### Валидация в Реальном Времени:
- ✅ **Имя**: минимум 2 символа
- ✅ **Контакт**: телефон (+996...) или email (name@email.com)
- ✅ **Сообщение**: минимум 10 символов
- ✅ **Компания**: опционально

#### Визуальные Индикаторы:
- 🟢 **Успех (field--success)**: зеленый бордер, значок ✓
- 🔴 **Ошибка (field--error)**: красный бордер, сообщение об ошибке
- 🎯 **Фокус (field--focused)**: акцентный цвет, увеличение на 2%

#### Помощные Подсказки:
- При фокусе на поле появляется подсказка
- Подпись переводится в активное состояние
- Ошибки отображаются под полем с анимацией

#### Состояния Отправки:
1. **Начальное**: "Отправить заявку"
2. **Отправка**: "Отправка..." с loading-анимацией
3. **Успех**: Всплывающее уведомление + очистка формы

**Пример валидации:**
```javascript
.field--success input { 
  border-color: #10b981;
  animation: glow 2s ease infinite; 
}
```

## 🎨 Ключевые Анимации

### Основные Keyframes:

```css
/* Появление с поднятием */
@keyframes fadeInUp {
  from { opacity: 0; transform: translateY(12px); }
  to { opacity: 1; transform: translateY(0); }
}

/* Слайд слева */
@keyframes slideInLeft {
  from { opacity: 0; transform: translateX(-20px); }
  to { opacity: 1; transform: translateX(0); }
}

/* Масштабирование */
@keyframes scaleIn {
  from { opacity: 0; transform: scale(0.95); }
  to { opacity: 1; transform: scale(1); }
}

/* Пульсирование */
@keyframes pulse {
  0%, 100% { opacity: 1; }
  50% { opacity: 0.6; }
}

/* Свечение */
@keyframes glow {
  0%, 100% { box-shadow: 0 0 5px rgba(22, 163, 74, 0); }
  50% { box-shadow: 0 0 15px rgba(22, 163, 74, 0.15); }
}
```

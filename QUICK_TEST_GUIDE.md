# 🚀 Быстрый Старт — Тестирование Анимаций

## Что было добавлено?

### ✨ Основные Функции:

1. **Анимации Кнопок & Карточек**
   - Гладкие hover-эффекты
   - Лифт и тень при наведении
   - Плавные переходы (300ms)

2. **Интерактивная Галерея**
   - Модальное окно при клике на фото
   - Стрелки для переключения изображений
   - Навигация клавиатурой (←/→ и ESC)

3. **Форма обратной связи с валидацией**
   - Проверка в реальном времени
   - Визуальные подсказки (зелёный/красный)
   - Уведомление об отправке

4. **Система уведомлений**
   - Успех (зелёный)
   - Ошибка (красный)
   - Автозакрытие через 5 сек

## 🧪 Как Тестировать?

### 1. Откройте index.html в браузере

### 2. Тест Анимаций Кнопок
```
✓ Наведите на кнопку "Получить консультацию"
  → Должна подняться на 2px и появиться тень
✓ Кликните на кнопку
  → Должна вернуться вниз (нажатие)
✓ Наведите на вторую кнопку
  → Другой стиль hover-эффекта
```

### 3. Тест Карточек
```
✓ Скролльте вниз на главной странице
  → Карточки появляются с fadeInUp анимацией
✓ Наведите на карточку "Чем мы полезны"
  → Поднимается на 4px, меняется цвет бордера
✓ Проверьте каскадное появление
  → Каждая карточка появляется с задержкой
```

### 4. Тест Галереи
```
✓ Нажмите на "Галерея" в меню или прокрутите вниз
✓ Кликните на любое изображение
  → Откроется модальное окно с увеличенным фото
✓ Нажмите стрелку "→" (следующее)
  → Показывается следующее изображение
✓ Нажмите стрелку "←" (предыдущее)
  → Показывается предыдущее изображение
✓ Нажмите клавишу ←
  → Переключение на клавиатуре
✓ Нажмите ESC
  → Модаль закрывается
✓ Нажмите на фон (серую область)
  → Модаль закрывается
✓ Нажмите X в правом верхнем углу
  → Модаль закрывается
✓ Проверьте счётчик слева
  → Показывает текущее фото и общее количество
```

### 5. Тест Формы
```
✓ Перейдите на "Контакты" или прокрутите вниз
✓ Наведите на поле "Имя"
  → Поле получит акцентный цвет
✓ Введите 1 символ
  → Должно появиться сообщение об ошибке (красная подсказка)
✓ Введите "Иван"
  → Поле станет зелёным, появится галочка ✓
✓ Введите некорректный email
  → Красная подсказка "Введите корректный email"
✓ Введите корректный email (name@email.com)
  → Станет зелёным с галочкой
✓ Введите телефон (+996...)
  → Проверка телефона +996...
✓ Кликните на "Отправить заявку"
  → Кнопка покажет "Отправка..." с анимацией
  → Через 1.5 сек форма очистится
  → Появится зелёное уведомление "✓ Спасибо! Ваша заявка принята"
```

### 6. Тест Уведомлений
```
✓ Отправьте форму (см. выше)
  → Появится зелёное уведомление в нижнем правом углу
✓ Кликните на уведомление
  → Оно исчезнет
✓ Ждите 5 секунд
  → Уведомление автоматически закроется
```

### 7. Тест на Мобильном (DevTools F12)
```
✓ Откройте DevTools (F12 или Cmd+Opt+I)
✓ Нажмите на иконку мобильного устройства
✓ Выберите устройство (iPhone SE, Pixel 4, iPad)
✓ Проверьте:
  ✓ Кнопки имеют минимум 44x44px
  ✓ Галерея показывает 1 колонку на мобильном
  ✓ Форма удобна для ввода
  ✓ Модальное окно соответствует размеру экрана
  ✓ Уведомления растягиваются на весь экран (минус отступы)
✓ Протестируйте на разных размерах (375px, 390px, 412px, 768px)
```

### 8. Тест Производительности
```
✓ Откройте DevTools → Performance tab
✓ Нажмите Record
✓ Наведите на разные элементы (кнопки, карточки)
✓ Откройте галерею модаль
✓ Нажмите Stop
✓ Проверьте:
  ✓ FPS остаётся высокой (60fps)
  ✓ Нет лагов или фризов
  ✓ Нет длинных Yellow/Red полос
```

### 9. Тест Навигации
```
✓ Нажмите на "Получить консультацию" в заголовке
  → Плавная прокрутка к форме обратной связи
✓ Нажмите на "Оставить заявку" в любом месте
  → Плавная прокрутка к форме
✓ Прокрутите вверх быстро
  → Sticky header остаётся видным
```

### 10. Тест Клавиатурной Навигации
```
✓ Нажмите TAB несколько раз
  → Фокус переходит по элементам (outline видна)
✓ В модальном окне нажмите TAB
  → Фокус остаётся в модали (не уходит за пределы)
✓ Нажмите ENTER на кнопке
  → Выполнится действие кнопки
```

### 11. Тест Доступности
```
✓ В браузере откройте DevTools → Accessibility
✓ Проверьте:
  ✓ Все кнопки имеют ARIA-labels
  ✓ Форма имеет правильные labels
  ✓ Изображения имеют alt-тексты
✓ Используйте браузерный reader для проверки
```

## 🐛 Возможные Проблемы & Решения

| Проблема | Решение |
|----------|---------|
| Анимации работают, но очень медленно | Проверьте GPU acceleration (DevTools Performance) |
| Модаль не закрывается на ESC | Убедитесь, что модаль в фокусе |
| Форма не валидируется | Проверьте консоль (F12) на ошибки |
| Уведомление не появляется | Проверьте z-index и overflow на body |
| На мобильном элементы слишком маленькие | Используйте zoom в DevTools |

## 💡 Советы для Лучшего Тестирования

1. **Используйте DevTools Throttling**
   - DevTools → Network → Slow 3G
   - Проверьте, что всё ещё работает

2. **Тестируйте на реальном устройстве**
   - Используйте `http://localhost:8000` (python -m http.server)
   - Откройте на реальном телефоне/планшете

3. **Используйте различные браузеры**
   - Chrome, Firefox, Safari, Edge
   - Мобильные браузеры также важны

4. **Проверьте производительность**
   - DevTools → Performance → Record
   - Убедитесь в стабильных 60 FPS

5. **Проверьте Accessibility**
   - DevTools → Accessibility
   - Используйте screen reader

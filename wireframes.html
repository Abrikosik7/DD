Повторяющиеся блоки (компоненты)

Header
  • Лого
  • Меню (5 пунктов)
  • Опциональный CTA (кнопка)
Footer
  • Ключевые ссылки
  • Контакты (кратко)
  • Юр.инфо (опционально)
Карточки
  • Услуга (название, 1–2 строки, CTA)
  • Кейс (результат, метрики)
  • Сотрудник (имя, роль)
CTA‑блок
  • Заголовок + краткий текст
  • Primary/secondary кнопки
  • Опционально: контакты/срок ответа

Перечень UI‑элементов (к дизайну)

Навигация / структура
  • Header: лого, пункты меню, активный пункт
  • Footer: ссылки, контакты
  • Оглавление‑якоря (страница ТЗ)
Кнопки
  • Primary / Secondary / Link
  • Disabled / Loading (состояния)
Формы
  • Input, Textarea, Checkbox (согласие)
  • Validation: error/help text
  • Success / Error сообщения после отправки
Состояния
  • Empty state (нет данных)
  • Error state (сбой загрузки/отправки)
  • 404 (желательно)
